# Security Policy

## Reporting a Vulnerability

This MCP server is officially maintained by OrcaRouter. Please report
security issues — whether in this package or in the OrcaRouter platform —
through the support channel at
[https://www.orcarouter.ai](https://www.orcarouter.ai).

Do not open public GitHub issues for security concerns.

## Supported Versions

`@orcarouter/mcp` is under active development on the `v0.x` line. Apply
the latest patch version for fixes.
